# clarity-in-action
Work Sample Portfolio — Product Documentation &amp; Technical Writing. These are curated examples for documentation for healthcare platforms. Includes UX content, knowledge base and process documentation. Each sample demonstrates clarity, accessibility, and measurable impact in cross‑functional product environments.

## Welcome to MkDocs

For full documentation visit [mkdocs.org](https://www.mkdocs.org).

### Commands

* `mkdocs new [dir-name]` - Create a new project.
* `mkdocs serve` - Start the live-reloading docs server.
* `mkdocs build` - Build the documentation site.
* `mkdocs -h` - Print help message and exit.
